# AboutRijeka

AboutRijeka je web aplikacija razvijena s ciljem objedinjavanja važnih informacija o gradu Rijeci na jednom mjestu. Aplikacija omogućuje stanovnicima i posjetiteljima brz i jednostavan pristup informacijama o vremenu, prometu, sportu, kulturi i drugim korisnim gradskim sadržajima.

## Opis projekta

Cilj projekta bio je razviti funkcionalnu web aplikaciju koja korisnicima pruža pregled najvažnijih informacija o gradu Rijeci kroz jedinstveno i intuitivno korisničko sučelje.

Umjesto pretraživanja više različitih web stranica, korisnici mogu na jednom mjestu pronaći:

- vremensku prognozu
- informacije o prometu
- sportske aktivnosti i događanja
- kulturne sadržaje
- raspored svetih misa
- muzeje i kulturne ustanove
- lokacije javnih slavina
- ostale korisne informacije o gradu

Projekt je izrađen u sklopu studija informatike s ciljem razvoja funkcionalne web aplikacije koja objedinjuje relevantne informacije za građane i posjetitelje grada.

# AboutRijeka

AboutRijeka is a web application developed to provide residents and visitors with useful information about the City of Rijeka through a single centralized platform. The application combines various city-related services and information, making them easily accessible through a simple and user-friendly interface.

## Project Overview

The goal of this project was to develop a functional web application that gathers important information about Rijeka and presents it in one place.

Instead of searching through multiple websites, users can access:

- Weather forecasts
- Traffic information
- Sports activities and events
- Cultural attractions and events
- Mass schedules
- Museums and cultural institutions
- Public water fountain locations
- Other useful city-related information

The project was developed as part of the Informatics study program to create a practical and functional city information platform.

## Technologies

- HTML5
- CSS3
- JavaScript

## Features

- Display of current weather forecasts for the City of Rijeka
- Traffic information overview
- Sports activities and event listings
- Cultural attractions and institution information
- Museum information
- Public water fountain locations
- Dynamic content presentation
- User-friendly interface

## Data Sources

The application uses publicly available data sources and presents relevant information to users in a centralized location.
